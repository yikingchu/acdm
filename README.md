# ACDM系统
  开源的ACDM系统，机场协同、CDM方案。  
## 功能实现
  现目前实现以下功能：  
  * 移动端保障；包括18个节点保障、紧急PUSH、重要监控、GIS管控等  
  * 物联网接入；包括跑道防入侵系统、质量安全控制系统、ACDM指挥大屏系统、塔台数字化系统、ATC监控监测系统等  
  * 电脑端总控；包括可视化数据平台、业务运行平台等
 
## 本开源项目开源内容  
   开源MUI混合开发的移动APP端。 
## 遵守原则
   开源项目不可应用于商业用途，同样也不可以修改后二次发布获取权利等。
